## 🤝 Contributing

We ❤️ contributions from the global developer community! Every pull request makes this hub more powerful for everyone.

```bash
# Get started in 60 seconds:

1. 🍴  Fork this repository on GitHub
2. 🌿  git checkout -b add/your-awesome-resource
3. ✅  Add your resource to the correct file in /resources/
4. 💬  git commit -m "add: [Collection] Resource Name — brief description"
5. 📤  git push origin add/your-awesome-resource
6. 🔃  Open a Pull Request — reviewed within 48 hours!
```

> 📌 **Our Quality Bar:** Resources must be actively maintained, genuinely useful, and clearly marked as free or paid.
> Duplicates and low-quality submissions will be closed with kindness. See [CONTRIBUTING.md](./CONTRIBUTING.md) for full guidelines.
